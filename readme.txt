== Description ==
Plugin for wordpress which gives a web-based graphical interface for filtering posts in a category, specifically a drag-and-drop interface for publishing/hiding posts.

== Installation ==
1. Upload the directory `postselector` to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
